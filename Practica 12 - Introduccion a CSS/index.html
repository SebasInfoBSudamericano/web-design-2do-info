Ejemplo de CSS:

CSS hace referecia a las hojas de estilo en cascada

Cuando hablamos de cascada, significa que la codificacion se lee de arriba a abajo, estableciendo el valor final de la ultima linea como el que prevalece de un mismo estilo codificado

Codificacion:

			input{
				color: red;
			}

input: Selector de Codigo

color: Atributo

red: Valor de Atributo
